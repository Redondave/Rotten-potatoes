# Rotten Potatoes - a rotten tomatoes-like movie listing website

This is a project built using the Ruby on Rails framework, implementing a MVC pattern to explore the possibilities of this architecture for the development of SaaS applications. For this project I'm using
# Ruby version
- Ruby 3.3.11 and Rails 8.1.3
# System dependencies
- Capybara and Cucumber with RSpec for testing, and SQLite for Models
# Configuration
- In order to run the application, you simply must follow the Rails setup guide, as avaliable in https://guides.rubyonrails.org/getting_started.html
- If needed, run "bundle install" to reolve any gem dependencies (if bundle not installed, install it using prefered package installator, such as apt or yarn)
- After that, simply launch the application using "rails server"
# Database initialization
- If needed, run "rails db:migrate" to update the Model to the latest changes
